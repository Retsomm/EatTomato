# 使用vite tailwind yarn

使用 Vite、Tailwind CSS 和 Yarn 來打造 React + Electron 版的番茄鐘應用，能加速開發並產生更輕量的 bundle。[^1][^2]

## 專案初始化

用 Yarn 建立 Vite + React 專案：`yarn create vite pomodoro-timer --template react-ts`，進入資料夾 `cd pomodoro-timer`，然後安裝依賴 `yarn`。[^4]
安裝 Electron 相關套件：`yarn add -D electron electron-builder vite-plugin-electron @vitejs/plugin-react`，並新增主進程檔案 `electron/main.ts` 建立 BrowserWindow。[^8]
安裝 Tailwind：`yarn add -D tailwindcss @tailwindcss/vite`，在 `vite.config.ts` 加入 `tailwindcss()` 外掛，並於 `src/index.css` 頂端加 `@import "tailwindcss";`。[^3][^4]

## 腳本與開發設定

修改 `package.json` scripts：

```
"scripts": {
  "dev": "concurrently \"yarn dev:react\" \"yarn dev:electron\"",
  "dev:react": "vite",
  "dev:electron": "wait-on http://localhost:5173 && electron .",
  "build": "vite build && electron-builder",
  "preview": "vite preview"
}
```

新增 `electron.vite.config.ts` 整合 Vite 外掛，開發時用 `yarn dev` 啟動熱重載。[^2][^1]

## UI 與功能開發

在 `src/App.tsx` 用 Tailwind 建置響應式介面：大時鐘顯示（`useEffect` + `Date`）、番茄鐘（`useState` 計時器 + `setInterval`）、計時器（自訂輸入）。[^3]
用 Tailwind 類別如 `bg-gradient-to-r from-blue-500 to-purple-600 text-6xl font-mono` 美化按鈕與計時畫面，支援深色模式 `dark:bg-gray-900`。[^1]

## 打包部署流程

執行 `yarn build` 先產生靜態檔案，再用 electron-builder 打包跨平台安裝檔（設定 `package.json` 的 `build` 區塊指定 `win`、`mac`、`linux`）。[^8]
產生 .exe/.dmg/.deb 檔案後，上傳 GitHub Releases，使用者下載雙擊安裝，首次開啟顯示托盤圖示與全螢幕時鐘。[^2]

## 使用者操作

安裝後啟動應用，介面切換時鐘/番茄/計時模式，按「開始」即計時並播放鈴聲通知，資料存 localStorage，下次開啟自動恢復進度。[^1]
